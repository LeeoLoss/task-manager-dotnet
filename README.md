# 🚀 Task Manager .NET

Um sistema de gerenciamento de tarefas (To-Do List) desenvolvido em **C#** e **.NET**, agora integrado totalmente à nuvem.

## 🛠️ Tecnologias Utilizadas

* **Linguagem:** C#
* **Framework:** .NET 8.0 (Console Application)
* **ORM:** Entity Framework Core
* **Banco de Dados:** Azure SQL Database (Microsoft Cloud)
* **Versionamento:** Git & GitHub

## ☁️ Evolução do Projeto

O projeto iniciou com persistência local utilizando **SQLite**, mas foi migrado para uma arquitetura de nuvem utilizando o **Azure SQL**. Isso permite que os dados sejam acessados de qualquer lugar, garantindo integridade e escalabilidade.
