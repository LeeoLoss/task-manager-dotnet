# 📝 Task Manager Console

Um gerenciador de tarefas simples e funcional desenvolvido em **C#** para consolidar conceitos de Programação Orientada a Objetos e CRUD em memória.

## 🚀 Funcionalidades
- **Adicionar Tarefas:** Título, descrição, prioridade e status.
- **Listar Tarefas:** Visualização de todos os itens cadastrados.
- **Editar/Excluir:** Gerenciamento completo de itens existentes.
- **Confirmação de Dados:** Lógica de validação antes de salvar.

## 🛠️ Tecnologias Utilizadas
- **Linguagem:** C#
- **Framework:** .NET 8.0/9.0
- **Persistência:** Em memória (Listas)
- **Consultas:** LINQ (FirstOrDefault)

## 📖 Como Rodar o Projeto
1. Certifique-se de ter o SDK do .NET instalado.
2. Clone o repositório: `git clone https://github.com/SEU_USUARIO/NOME_DO_REPO.git`
3. Entre na pasta e execute: `dotnet run`
